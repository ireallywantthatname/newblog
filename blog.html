Inheritance in Java

In inheritance, we can create a new class using an existing class. In other words, inheritance is a type of relationship. Once you create a class to inherit from a parent class, the child class can use the features of the parent class.

Syntax

class ParentClass {
  //......
}

class ChildClass extends ParentClass {
  //......
}

In inheritance, the child object can access the parent class non-private data or methods.

Constructors in Inheritance

Once you declare a data member or method as protected in a base class, it can be accessed only inside the immediate child class.
